    Help

    使用方法

    1. 粘贴客户问题。
    2. 可选填写商品名、SKU 或型号，越具体越容易命中。
    3. 点击“开始查询”，等待飞书检索和回复生成完成。
    4. 查看文档信息、命中来源和建议回复。
    5. 如果查询不准，展开“执行详情”查看检索关键词，再补充更精确的商品信息。
    6. 使用模型回复前，请先在设置里配置 Ollama 或 OpenAI Compatible。
    7. 飞书 CLI：另一台电脑需要先安装 lark-cli，并先运行 lark-cli auth login 完成基础登录。
    8. 飞书 CLI：如果文档搜索提示缺少权限，请运行 lark-cli.cmd auth login --scope "search:docs:read" 补充搜索授权。
    9. 飞书 CLI：用 lark-cli docs +search --as user --query "咖啡机" --format json 验证能搜索文档。
    10. 如果执行详情里出现 need_user_authorization，说明授权过期，需要重新运行 lark-cli auth login。
